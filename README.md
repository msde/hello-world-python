# hello world python

scaffolding for super simple python app.  To run tests, `make deps test`.  

To run, `make deps run`.

## design notes

See `docs/design.md`

## initial system setup (macos)

```
brew install pyenv
pyenv install 3.8.2
pyenv global 3.8.2
```

## installation
```
make deps
```

## running main.py
```
make run
```

## running tests
```
make test
```
